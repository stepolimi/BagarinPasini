{The goals of \hbox{\emph{TrackMe}} are the following:
\vspace{0.3cm}

\begin{itemize}
	\item[${[G1]}$] {Allows the user to register and to have a personal area to which he/she can access with a secure code.}
	\item[${[G2]}$] {Allows the third party to register and to have a personal area to which it can access with a secure code.}
	\item[${[G3]}$] {Allows the third party to require data.
				\begin{itemize}
					\item[${[G3.1]}$] {Third party can require single person's data.}
					\item[${[G3.2]}$] {Third party can require data of group of people.}
				\end{itemize}}
	\item[${[G4]}$] {Allows the user to accept or not to let a third party to have access to his/her data.}
	\item[${[G5]}$] {Allows third party to subsribe for new data.}
	\item[${[G6]}$] {Allows third party to obtain the required data if the request has been accepted, a notification if it has been refused.}
	\item[${[G7]}$] {Allows the user to receive quick help if his/her vital parameters are lower then treshold values.}

	
\end{itemize}
